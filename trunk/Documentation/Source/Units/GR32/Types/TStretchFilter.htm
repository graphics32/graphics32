type TStretchFilter = (sfNearest, sfDraft, sfLinear, sfCosine, sfSpline, sfLanczos, sfMitchell);

Description

When the bitmap is scaled, these filters control how the colors are interpolated.

In some functions filters above sfLinear may not be applicable, in this case the filter value will be substituted with sfLinear.

[See Also] TBitmap32.StretchFilter, StretchTransfer
