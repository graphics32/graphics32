constructor Create(ColorGradient: TColor32Gradient); overload; override;

constructor Create(LookupTable: TColor32LookupTable); overload; virtual;

Description

Creates and initializes an instance of TLinearGradientLookupTablePolygonFiller.

Either a TColor32Gradient object or a TColor32LookupTable object can be passed to the constructor, which is used to draw the color gradient. In both cases the passed objects are configured accordingly in the constructor. This includes modifying the OnGradientColorsChanged event of the passed TColor32Gradient object
