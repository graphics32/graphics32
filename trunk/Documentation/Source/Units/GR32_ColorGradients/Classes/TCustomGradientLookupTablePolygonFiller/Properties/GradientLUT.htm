property GradientLUT: TColor32LookupTable; //read and write

Description

The property GradientLUT points to the look up table (TColor32LookupTable), which is used by the polygon filler, in case if it is present. An existing lookup table is freed in case it is owned internally. If nil is passed, UseLookupTable is set to false. If an internal look up table shall be used again, UseLookupTable must then be set to true again.
