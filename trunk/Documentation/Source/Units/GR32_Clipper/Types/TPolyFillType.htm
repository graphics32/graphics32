TPolyFillType = (pftEvenOdd, pftNonZero, pftPositive, pftNegative);


Description

  
  

Graphics32 currently only supports the two most widely used polygon filling rules for polygon rendering - EvenOdd (aka Alternate) and NonZero (aka Winding). However, while Positive and Negative filling rules are not supported by the renderer, the TClipper class used in clipping operations does accommodate them (see TClipper.Execute).
