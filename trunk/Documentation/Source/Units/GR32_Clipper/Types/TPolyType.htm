TPolyType = (ptSubject, ptClip);


Description

A Polygon or group of polygons added to a TClipper object can be either Subject or Clip polygons.

For Intersection, Union and XOR clipping operations it doesn't matter which group of polygons are assigned to which polygon type as the result will be the same. However, for Difference operations, the result polygons will be the subject polygons minus any regions covered by clip polygons.
