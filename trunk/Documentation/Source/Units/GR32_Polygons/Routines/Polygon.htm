procedure PolygonTS(Bitmap: TBitmap32; const Points: TArrayOfFixedPoint; Color: TColor32; Mode: TPolyFillMode = pfAlternate);

procedure PolygonAS(Bitmap: TBitmap32; const Points: TArrayOfFixedPoint; Color: TColor32; Mode: TPolyFillMode = pfAlternate);

procedure PolygonXS(Bitmap: TBitmap32; const Points: TArrayOfFixedPoint; Color: TColor32; Mode: TPolyFillMode = pfAlternate);

Description

Fills the shape defined by the Points parameter with specified color. Unlike GDI polygons, this function allows for transparency and optional antialiasing.

PolygonTS draws a non-antialiased polygon with optional transparency. When drawing, fixed-point coordinates of vertixes are rounded.

PolygonAS draws an antialiased polygon, fixed-point coordinates of vertexes are rounded.

PolygonXS provides full antialiasing and does not perform rounding of vertex coordinates.

All functions perform clipping.
