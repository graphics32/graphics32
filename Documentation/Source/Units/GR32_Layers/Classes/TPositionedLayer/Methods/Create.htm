constructor Create(ALayerCollection: TLayerCollection); override;

Description

Creates the instance of TPositionedLayer.

After calling the inherited constructor, Create initializes the following properties:

  • LayerOptions to LOB_VISIBLE or LOB_MOUSE_EVENTS;
  • Scaled to False;
  • Location to FloatRect(0, 0, 64, 64);
