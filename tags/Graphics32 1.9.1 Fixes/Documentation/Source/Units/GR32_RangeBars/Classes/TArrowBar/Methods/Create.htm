constructor Create(AOwner: TComponent); override;

Description

Creates and initializes an instance of TArrowBar.

After calling the inherited constructor, Create initializes the following properties:

  • Backgnd to bgPattern;
  • ButtonSize to 0;
  • Color to clBtnShadow;
  • Style to rbsDefault;
  • HandleColor to clBtnShadow;
  • Kind to sbHorizontal;
  • ShowArrows to True;
  • ShowHandleGrip to True;
