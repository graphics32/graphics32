procedure ApplyLUT(Dst, Src: TBitmap32; const LUT: TLUT8);

Description

Converts color from Src to Dst using a look-up table (LUT parameter) to map the color components. The alpha channel of the result is automatically set to $FF:

RDST = LUT[RSRC];

GDST = LUT[GSRC];

BDST = LUT[BSRC];

ADST = $FF;

This function supports in-place operation, that is Dst may be the same as Src.
