constructor Create(ALayerCollection: TLayerCollection); virtual;

Description

Creates an instance of TCustomLayer and adds it to the specified layer collection, for example:

var

  MyLayer: TCustomLayer;

begin

  MyLayer := TCustomLayer.Create(MyImage32.Layers);

  ...

end;

By default custom layers are created with only LOB_VISIBLE bit in their LayerOptions.

[See also]: Using Layers
